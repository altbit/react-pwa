# React Progressive Web App #

webpack 2.4+
react 15.5+
react-router 4.1+
material-ui 1.0+ (alpha yet)
redux
redux-form
express
mongoose
server node v6+ (ES6)

### Install Node.js + NPM ###

https://nodejs.org/en/download/

### Install Node modules ###

$ npm install

### Build project ###

$ npm run build

### Run Server API ###

$ DEBUG=server*,router*,controller*,service* npm start

### Run dev server (development) ###

$ npm run dev
